Overview

Remus is a low-level C library providing one-sided RDMA primitives for high-performance networking. It exposes APIs to register memory regions, perform RDMA reads/writes, and manage completion queues with minimal overhead.

Installation

git clone https://github.com/youruser/remus.git
cd remus
mkdir build && cd build
cmake ..
make -j$(nproc)
sudo make install

Usage

#include <remus/remus.h>

int main() {
    remus_context_t ctx;
    remus_init(&ctx);
    // ... your RDMA logic ...
    remus_cleanup(&ctx);
    return 0;
}

API Reference

  • int remus_init(remus_context_t *ctx) – initialize context.
  • int remus_reg_mr(remus_context_t *ctx, void *addr, size_t len, int access) – register a memory region.
  • int remus_post_send(remus_context_t *ctx, remus_wr_t *wr) – post an RDMA read/write.
  • int remus_poll_cq(remus_context_t *ctx, int timeout_ms) – poll completion queue.

Examples

remus_wr_t wr = {
    .opcode      = REMUS_RDMA_READ,
    .local_addr  = local_buf,
    .remote_addr = remote_addr,
    .length      = size
};
remus_post_send(&ctx, &wr);
remus_poll_cq(&ctx, 1000);
